# Roadmap

## Near Term

- Add optional local model provider support.
- Add calendar/reminder integrations.
- Add richer conversation history controls.

## Mid Term

- Build iPhone-native wrapper/client with same local-first principles.
- Add configurable notification rules and daily planning workflows.
- Add usage analytics dashboard from local data.

## Long Term

- Pluggable tool framework for custom user workflows.
- Fine-grained permission model for sensitive actions.
- More advanced personalization with explicit user approval controls.
