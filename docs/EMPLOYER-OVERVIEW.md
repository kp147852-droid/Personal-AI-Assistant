# Employer Overview

## Project Summary

Alpha is a local-first AI assistant prototype that combines conversation, image explanation, and actionable coaching with a security-aware architecture.

## What This Demonstrates

- Product thinking: solution built around real user pain points (comprehension, focus, task follow-through).
- AI integration: multimodal and conversational interfaces with resilient fallback behavior.
- Data fluency: structured local persistence to support future analytics and personalization.
- Delivery: end-to-end implementation from API design to UX and runtime ops.

## Resume Bullet Ideas

- Built a local-first AI assistant (FastAPI + JS frontend) with chat, voice, and image explanation workflows.
- Implemented single-user security controls (localhost binding, host restrictions, request-origin constraints).
- Designed user-centered ADHD coaching flows with task and check-in persistence in SQLite.
- Improved reliability with startup/runtime safeguards and normalized AI error handling.

## Interview Talking Points
